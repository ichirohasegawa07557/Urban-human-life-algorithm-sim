# Scientific Scope

This project is an agent-based urban-life digital twin simulation.

## Causal chain

```text
urban layout
  -> resident routines
  -> mobility / congestion
  -> fatigue / satisfaction
  -> energy / waste / spending / public-service load
  -> city-scale outcomes
```

## Included components

- synthetic residents
- roles: worker / student / senior
- homes, work, schools, stations, commerce, hospitals, parks, admin
- morning and evening commuting
- lunch and commerce behavior
- hospital and park access
- congestion heatmap
- fatigue and satisfaction
- energy demand
- waste generation
- public-service load
- policy counterfactuals

## Not included

- real personal data
- real GPS traces
- calibrated transit model
- real economic forecast
- legal / official policy recommendation

This is a conceptual and educational simulation.
